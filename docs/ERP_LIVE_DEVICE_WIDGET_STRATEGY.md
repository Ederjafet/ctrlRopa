# ERP LIVE - Estrategia de widgets por dispositivo

Fecha: 2026-05-20  
Fase: LIVE-V

## Desktop

Desktop puede mostrar mas widgets porque se usa para supervision, demo comercial y lectura amplia:

- spotlight producto,
- vista presentadora,
- estado operativo,
- roles,
- analiticos,
- actividad.

## Tablet

Tablet prioriza operacion:

- producto,
- reserva,
- reservas recientes,
- estado operativo.

Roles, analiticos y actividad respetan preferencias, pero deben mantenerse compactos.

## Mobile

Mobile es una vista rapida:

- producto si esta activo,
- flujo de reserva,
- reservas recientes.

Por defecto se ocultan automaticamente:

- roles,
- analiticos,
- actividad.

## QA

Validar que ocultar widgets no deja huecos, no rompe scroll y mantiene visibles las acciones principales.

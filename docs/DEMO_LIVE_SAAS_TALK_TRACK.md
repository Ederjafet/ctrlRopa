Fecha: 2026-05-18

## Apertura Corta

"HPSQ-SOFT esta evolucionando este sistema hacia un ERP SaaS multiempresa. La base ya demuestra aislamiento por empresa en clientes, inventario y lotes; y LIVE commerce muestra el diferenciador hacia venta moderna, engagement y futura integracion con redes sociales."

## Version para Cliente Potencial

"La idea es que tu operacion no dependa de hojas sueltas, mensajes o memoria del vendedor. Desde una sola plataforma puedes controlar clientes, prendas, lotes y sesiones live. A futuro, LIVE no solo sera captura; tambien medira interes, comentarios, reacciones y productos destacados."

"Esta demo no es una promesa de produccion completa. Es una narrativa controlada: tenant-aware ya avanza en customers/items/batches, LIVE ya tiene base i18n, UX normalizada y metricas demo. El siguiente paso real no es vender humo, sino cerrar smoke visual, eventos internos y metricas reales."

## Preguntas Frecuentes

**Ya esta listo para produccion SaaS?**  
No completo. Hay avances reales en tenant-aware para customers/items/batches, pero faltan ventas, pagos, live completo, reportes y permisos por company.

**Las metricas LIVE son reales?**  
No. Son demo visual para presentar la experiencia futura.

**Se conecta a Facebook?**  
Todavia no. Ya existe diseno de integracion futura, pero no runtime productivo.

**Que modulo esta mas maduro para mostrar?**  
Customers, items, batches tenant-aware y LIVE visual/i18n/demo.

**Que sigue?**  
Smoke visual final, eventos internos LIVE, metricas reales internas, integracion Facebook, y luego reservaciones/ventas/pagos/reportes por fases.

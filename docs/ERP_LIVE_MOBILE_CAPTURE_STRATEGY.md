# ERP LIVE - Estrategia mobile de captura

Fecha: 2026-05-20  
Fase: LIVE-W

## Objetivo

Reducir saturacion en mobile y mejorar ergonomia tactil.

## Ajustes

- Mas espacio vertical entre bloques de cliente y prenda.
- Acciones principales ocupan el foco visual.
- Acciones de creacion se presentan como terciarias.
- Botones secundarios quedan agrupados bajo preguntas claras.

## QA recomendado

Validar en Android:

- input de codigo/QR visible,
- CTA `Agregar prenda` inmediato,
- bloque `¿No tienes codigo?` sin empalmes,
- `Reservar ahora` visible despues de precio,
- sin botones cortados ni wraps rotos.
